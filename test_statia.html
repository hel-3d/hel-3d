Texture_script for UE5

Tech Stack: python unreal_engine_api editorassetlibrary materialeditinglibrary
Plugin for automated texture importing.
This script automates asset preparation in Unreal Engine 5. It enables bulk import of texture maps, automatically organizes them into folders (creating new ones as needed), and generates a unique material for each texture set.

Key Features

  • Auto-create folder structure for textures and materials
  • Bulk import of texture maps
  • Auto-sorting of textures into folders
  • Auto-generation of materials for each texture set

Code Example

import unreal

def import_textures_and_create_materials(texture_dir, material_dir):
    # Example: imports all textures from a folder, creates materials for each
    asset_tools = unreal.AssetToolsHelpers.get_asset_tools()
    texture_assets = asset_tools.import_assets(texture_dir, '/Game/Textures/')
    for texture in texture_assets:
        mat_name = f"MAT_{texture.get_name()}"
        material = unreal.MaterialFactoryNew()
        asset_tools.create_asset(mat_name, material_dir, None, material)
    print("Import and material creation complete.")
