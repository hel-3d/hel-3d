Скрипты для ускорения работы

convert_to_pdf.py

  • Быстро создаёт pdf файл из всех изображений в выбранной папке согласно алфавитному порядку.
  • При создании pdf выравнивает размер изображений по первому, присваивает новому файлу название первого изображения.
  • Скрипт работает с изображениями в форматах: png, jpg, jpeg, bmp, gif, tiff.
  • Удобный интерфейс для использования скрипта.
Перейти на страницу проекта Перейдите на страницу проекта для бесплатного скачивания convert_to_pdf.py

convert_pdf_to_images.py

  • Быстро нарезает pdf файл на страницы и сохраняет в выбранную папку в выбранном формате (JPG, PNG, TIFF).
  • Каждое изображение получает имя в формате [название файла]_page_[номер страницы].[формат].
  • Удобный интерфейс для использования скрипта.
